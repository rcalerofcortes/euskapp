⚠️ Problema Detectado

No puedes abrir index.html haciendo doble clic en el archivo. Los navegadores modernos bloquean JavaScript por seguridad cuando usas el protocolo file://

🔍 Por qué no funciona:
Los archivos JavaScript no se pueden cargar desde file:///C:/... por restricciones de CORS (Cross-Origin Resource Sharing).

✅ Soluciones (Elige una):

🚀 Método 1: Usar el archivo START_SERVER.bat (MÁS FÁCIL)

  1. Busca el archivo START_SERVER.bat en esta carpeta
  2. Haz doble clic en START_SERVER.bat
  3. Se abrirá una ventana negra (no la cierres)
  4. Abre tu navegador en: http://localhost:8000
  5. ¡Listo! Ya verás la pantalla de login
✅ Recomendado: Esta es la forma más fácil si tienes Python instalado.

🎨 Método 2: Usar Live Server en VS Code

Si estás usando Visual Studio Code:

  1. Instala la extensión "Live Server" de Ritwick Dey
  2. Abre la carpeta euskapp en VS Code
  3. Clic derecho en index.html
  4. Selecciona "Open with Live Server"
  5. Se abrirá automáticamente en tu navegador

🐍 Método 3: Python Manual

Si tienes Python instalado:

  1. Abre Terminal o CMD
  2. Navega a la carpeta: cd C:\Users\29851\repos\euskapp
  3. Ejecuta: python -m http.server 8000
  4. Abre: http://localhost:8000

🌐 Método 4: Node.js (si lo tienes instalado)

  1. Abre Terminal/CMD en esta carpeta
  2. Ejecuta: npx serve .
  3. Abre la URL que te muestre
